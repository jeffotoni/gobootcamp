go build Plugins

A partir do Go 1.8, há um novo sistema de plug-in Go. Este recurso permite que os programadores criem programas modulares fracamente acoplados usando pacotes compilados como bibliotecas de objetos compartilhados que podem ser carregados e vinculados dinamicamente em tempo de execução.

Você pode conferir aqui alguns comentários: Go Plugin

Veja o exemplo ao lado e também o exemplo a baixo (lower.go):


// Go in action
// @jeffotoni
// 2019-01-18

package main

import (
  "strings"
)

type lower string

func (t lower) MustLower(name string) string {

  return strings.ToLower(name)
}
                        

$ go build -buildmode=plugin -o lower.so lower.go

$ go run main.go
