Editores - NeoVim

Editor de texto hiperextensível baseado em Vim

Entre seus recursos estão:

  • A API é de primeira classe: detectável, com versão, documentada.
  • A comunicação estruturada MessagePack permite extensões em qualquer idioma.
  • Os plug-ins remotos são executados como co-processos, de forma segura e assíncrona.
  • GUIs, IDEs, navegadores da web podem - incorporar o Neovim como um editor ou host de script.
