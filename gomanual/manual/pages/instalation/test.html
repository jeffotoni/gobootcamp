Instalação - Testando a Instalação

Vamos rodar a versão go para ver se está tudo correto: 


$ go version
go version go1.11.4 linux/amd64

Verifique se o Go está instalado corretamente configurando um espaço de trabalho e criando um programa simples, como segue.

Crie o diretório do seu espaço de trabalho, $HOME/go. (Se desejar usar um diretório diferente, você precisará definir a variável de ambiente $GOPATH.)

Em seguida, crie o diretório src/hello dentro de sua área de trabalho e, nesse diretório, crie um arquivo chamado hello.go que se pareça com:

  Instalação - $GOPATH Instalação - Workspace
