String

Um string type representa o conjunto de string values. Um string value é uma sequência (possivelmente vazia) de bytes. Strings são imutáveis: uma vez criadas, é impossível alterar o seu conteúdo.

O comprimento de uma string (seu tamanho em bytes) pode ser descoberto usando a função interna len(). O comprimento é uma constante de tempo de compilação se a string for uma constante. Os bytes de uma string podem ser acessados por índices inteiros de 0 a len(s) - 1. É ilegal obter o endereço de tal elemento; se s[i] é o i'ésimo byte de uma string, &s[i] é inválido.
