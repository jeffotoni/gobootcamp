Tipos - Slice

Um slice é um descritor para um segmento contíguo de um array subjacente e fornece acesso a uma sequência numerada de elementos desse array. Um tipo de slice denota o conjunto de todas as fatias de arrays de seu tipo de elemento. O valor de um slice não inicializada é nil.

Um array tem um tamanho fixo. Um slice, por outro lado, é uma visão flexível e de tamanho dinâmico dos elementos de um array. Na prática, os slices são muito mais comuns do que os arrays.

Um slice é formad especificando dois índices, um limite inferior e um limite superior, separados por dois pontos:

O tipo []T é um slice com elementos do tipo T. 


a[primeiro:último]

Isso seleciona um intervalo semi-aberto que inclui o primeiro elemento, mas exclui o último.

Um novo valor de slice inicializado para um determinado tipo de elemento T é feito usando a função embutida make, que pega um tipo de slice e parâmetros especificando o comprimento e, opcionalmente, a capacidade. Um slice criado com make sempre aloca um novo array oculto ao qual o valor do slice retornado se refere. 


make([]T, length, capacity)

Como os arrays, os slices são sempre unidimensionais, mas podem ser compostos para construir objetos de dimensões superiores. Com arrays de arrays, os arrays internos são, por construção, sempre do mesmo comprimento; no entanto, com slices de slices (ou arrays de slices), os comprimentos internos podem variar dinamicamente. Além disso, os slices interns devem ser inicializados individualmente.
