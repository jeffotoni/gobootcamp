Hello, World!

Vamos aprender como imprimir dados na tela - que na verdade é a saída padrão stdout, usando as funções print, println e fmt.Println.

As implementações atuais fornecem várias funções integradas úteis durante a inicialização. Essas funções são documentadas para integridade, mas não têm garantia de permanência no idioma. Eles não retornam um resultado.

Restrição de implementação: print e println não precisam aceitar tipos de argumentos arbitrários, mas a impressão de tipos booleanos, numéricos e de string deve ser suportada.

println é uma função embutida (no tempo de execução) que pode eventualmente ser removida, enquanto o pacote fmt está na biblioteca padrão, que irá persistir.

fmt.Println é uma função da lib "fmt" uma biblioteca padrão Go, ela é bem completa e bem extensa. Ela irá possuir diversas funcionalidades para saída. 


Função     Comportamento

print          Imprime todos os argumetos; a formatação dos argumentos é específica 
de cada implementação.

println        Parecido com print, mas separa os argumentos com um espaço 
e adiciona uma quebra de linha no final.

fmt.Println    É uma lib mais completa, irá possuir diversas funções para manipulação
de saída e seus respectivos tratamentos. Algumas delas: fmt.Print, fmt.Printf, fmt.Println, 
fmt.Sprintln, fmt.Sprintf, fmt.Fprint etc...
