Declaração de Variáveis

A declaração de uma variável cria uma ou mais variáveis, ataxando identificadores correspondentes a elas, e atribuindo a cada uma um tipo e um valor inicial.

Se uma lista de expressões é fornecida, as variáveis são inicializadas com as expressões seguindo as regras para atribuições. Caso contrário, cada variável é inicializada com seu valor zero.

Se um tipo estiver presente, cada variável receberá esse tipo. Caso contrário, cada variável recebe o tipo do valor de inicialização correspondente na atribuição. Se esse valor for uma constante sem tipo, ele será primeiro convertido para seu tipo padrão; se for um valor booleano sem tipo, primeiro será convertido para o tipo bool. O valor pré-declarado nil não pode ser usado para inicializar uma variável sem tipo explícito.

Restrição de implementação: Um compilador pode tornar ilegal declarar uma variável dentro do corpo de uma função se a variável nunca for usada.
