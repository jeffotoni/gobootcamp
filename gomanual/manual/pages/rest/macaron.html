API REST - Macaron

Pacote macaron é um framework web modular e altamente produtivo em Go.

  • Roteamento poderoso com suburl.
  • Combinações de rotas flexíveis.
  • Roteadores de grupo aninhados ilimitados.
  • Integre-se diretamente com os serviços existentes.
  • Altere dinamicamente os arquivos de modelo no tempo de execução.
  • Permitir o uso de modelos na memória e arquivos estáticos.
  • Recursos fáceis de conectar / desconectar com design modular.
  • Melhor camada de roteador e menos reflexão tornam a velocidade mais rápida.

package main

import "gopkg.in/macaron.v1"

func main() {
    m := macaron.Classic()
    m.Get("zerohero/", func() string {
        return "Hello world!"
    })
    m.Run()
}
