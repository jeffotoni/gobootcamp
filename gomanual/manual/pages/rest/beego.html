API REST - Beego

FÁCIL DE USAR

Com suporte RESTful, modelo MVC e uso de ferramenta Bee para construir seus aplicativos rapidamente com recursos que incluem compilação a quente de código, teste automatizado e empacotamento e implantação automatizados.

INTELIGENTE

Com roteamento e monitoramento inteligentes, é capaz de monitorar seu QPS, uso de memória e CPU e status de goroutine. Ele fornece a você controle total de seus aplicativos online.

MODULAR

Com poderosos módulos integrados, incluindo controle de sessão, armazenamento em cache, registro, análise de configuração, supervisão de desempenho, manipulação de contexto, suporte ORM e simulação de solicitações. Você obtém a base poderosa para qualquer tipo de aplicativo.

ALTA PERFORMANCE

Com pacote nativo Go http para lidar com as solicitações e a concorrência eficiente de goroutine. Seus aplicativos do beego podem lidar com o tráfego massivo, como o beego está fazendo em muitas produções.

beego.go

package main

import (
    "github.com/beego/beego/v2/server/web"
)

type MainController struct {
    web.Controller
}

func (this *MainController) Get() {
    this.Ctx.WriteString("hello world")
}

func main() {
    web.Router("zerohero/", &MainController{})
    web.Run()
}
