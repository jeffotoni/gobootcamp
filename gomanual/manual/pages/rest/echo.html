API REST - Echo

Echo é uma microestrutura PHP semelhante ao Lumen. Ele tem muitos recursos e está bem documentado para criar aplicativos da web rápidos e fáceis com o GO. Você pode criar uma API restful sem usar uma estrutura. no entanto, as estruturas têm seu próprio conjunto de recursos que tornam o desenvolvimento da web mais seguro e rápido.

  • Roteador HTTP altamente otimizado com zero alocação de memória dinâmica que prioriza rotas de forma inteligente.
  • Construa uma API RESTful robusta e escalonável, facilmente organizada em grupos.
  • Instalar automaticamente certificados TLS do Let's Encrypt.
  • O suporte HTTP / 2 melhora a velocidade e fornece uma melhor experiência do usuário.
  • Muitos middleware embutido para usar ou definir o seu próprio. O middleware pode ser definido em nível de raiz, grupo ou rota.
  • Vinculação de dados para carga útil de solicitação HTTP, incluindo JSON, XML ou dados de formulário.
  • API para enviar uma variedade de respostas HTTP, incluindo JSON, XML, HTML, Arquivo, Anexo, Inline, Fluxo ou Blob.

echo.go

package main

import (
	"net/http"
	
	"github.com/labstack/echo/v4"
)

func main() {
	e := echo.New()
	e.GET("/v1/api", func(c echo.Context) error {
		return c.String(http.StatusOK, "GETTING...")
	})

    e.POST("/v1/api", func(c echo.Context) error {
		return c.String(http.StatusOK, "POSTING...")
	})

    e.PUT("/v1/api", func(c echo.Context) error {
		return c.String(http.StatusOK, "PUTTING...")
	})

    e.DELETE("/v1/api", func(c echo.Context) error {
		return c.String(http.StatusOK, "DELETING...")
	})
	e.Logger.Fatal(e.Start(":8000"))
}
