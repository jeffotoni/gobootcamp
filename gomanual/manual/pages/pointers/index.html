Ponteiros - Introdução

Ponteiro é uma variável usada para armazenar o endereço de memória de outra variável. Ponteiros em Golang também são chamados de variáveis ​​especiais. As variáveis são usadas para armazenar alguns dados em um endereço de memória específico no sistema. O endereço de memória é sempre encontrado em formato hexadecimal (começando com 0x como 0xFFAAF etc.).

Para entender essa necessidade, primeiro, temos que entender o conceito de variáveis. Variáveis ​​são os nomes dados a um local da memória onde os dados reais são armazenados. Para acessar os dados armazenados, precisamos do endereço desse local de memória específico. Lembrar todos os endereços de memória (formato hexadecimal) manualmente é uma sobrecarga, por isso usamos variáveis ​​para armazenar dados e as variáveis ​​podem ser acessadas apenas usando seu nome.

Golang também permite salvar um número hexadecimal em uma variável usando a expressão literal, ou seja, o número que começa em 0x é um número hexadecimal.

Por exemplo, digamos que estamos armazenando o número hexadecimal em uma variável. Mas você pode ver que o tipo de valor é int e salvo como decimal ou pode dizer que o valor decimal do tipo int está armazenando. Mas o ponto principal para explicar este exemplo é que estamos armazenando um valor hexadecimal (considere-o um endereço de memória), mas não é um ponteiro, pois não está apontando para qualquer outro local de memória de outra variável. É apenas uma variável definida pelo usuário. Portanto, surge a necessidade de ponteiros.

Um ponteiro é um tipo especial de variável que não é usado apenas para armazenar os endereços de memória de outras variáveis, mas também aponta onde a memória está localizada e fornece maneiras de descobrir o valor armazenado naquele local da memória. Geralmente é denominado como um tipo especial de variável porque é quase declarado como uma variável, mas com * (operador de desreferenciamento).

existem dois operadores importantes que usaremos em ponteiros:

  • * Operador também denominado como o operador de desreferenciamento usado para declarar a variável de ponteiro e acessar o valor armazenado no endereço.
  • & operador denominado como operador de endereço usado para retornar o endereço de uma variável ou para acessar o endereço de uma variável para um ponteiro. 

var s *string // Variável do tipo ponteiro de string, ou seja, aceita somente endereços de variáveis do tipo string
