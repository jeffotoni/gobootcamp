Data e Hora - Duration

Duração é o tempo decorrido entre dois instantes de tempo. É representado como uma contagem de int64 nanosegundos. Portanto, a duração é apenas um número que representa o tempo em nanossegundos em Go.

Isso é representado da seguinte forma no pacote 'time': 


type Duration int64

Abaixo estão algumas das durações comuns no pacote: 


const (
    Nanosecond  Duration = 1
    Microsecond          = 1000 * Nanosecond
    Millisecond          = 1000 * Microsecond
    Second               = 1000 * Millisecond
    Minute               = 60 * Second
    Hour                 = 60 * Minute
)
