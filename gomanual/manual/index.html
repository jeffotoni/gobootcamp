Go Bootcamp

Este material tem o objetivo de contribuir para o aprendizado da linguagem de programação Go, colaborando com a comunidade como uma fonte de aprendizado da linguagem. O material aborda o que precisamos saber para iniciarmos a programação na linguagem Go.

O conteúdo visa tornar a vida de quem está iniciando na linguagem mais fácil.

Se você conhece pouco ou quase nada da programação não será problema, todo o manual foi feito para entusiastas do nível inicial ao intermediário.

Uma boa leitura.
